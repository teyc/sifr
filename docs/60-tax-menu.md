# 60-tax-menu.md - Tax Menu Summary

## Overview
The tax menu handles tax calculations, codes, and compliance reporting.

## Key UI Elements
- **Tax Codes/Rates**: Configurable list with jurisdiction-specific rates.
- **BAS/GST Reports**: Forms for filing with auto-calculations.
- **Tax Periods**: Calendar view for tracking obligations.
- **Filing Status**: Indicators for completed/submitted filings.

## Functional Workflows
- **Tax Calculation**: Automatic application to transactions.
- **Report Generation**: Pull data for tax authorities.
- **Reminders**: Alerts for upcoming deadlines.

## Accounting Principles
- Ensures accurate tax reporting per regulations.
- Audit trails for tax adjustments.

## Technical Considerations
- Jurisdiction-specific logic.
- Integration with tax APIs.

## AI Opportunities
- Tax code prediction.
- Compliance checks.

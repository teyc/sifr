# 50-accounting-menu.md - Accounting Menu Summary

## Overview
The accounting menu covers core bookkeeping, including chart of accounts, bank reconciliation, and manual journals.

## Key UI Elements
- **Chart of Accounts**: Hierarchical tree view with account types (assets, liabilities, etc.).
- **Reconciliation Interface**: Bank statement matching with transaction lists.
- **Manual Journals**: Form for adjusting entries.
- **Bank Accounts**: List with balances and reconciliation status.

## Functional Workflows
- **Reconciliation**: Match bank transactions to ledger entries.
- **Account Management**: Add/edit accounts with proper classification.
- **Journal Entries**: Record adjustments with double-entry validation.

## Accounting Principles
- Enforces double-entry bookkeeping.
- Immutable audit logs for all changes.

## Technical Considerations
- Real-time bank sync.
- Validation for balanced entries.
- Multi-tenant isolation.

## AI Opportunities
- Automated matching and categorization.
- Fraud detection.
